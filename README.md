# Dji-sdk-example

DJIのMSDK及びUXSDKを組み込んだサンプルプログラム

ブランチ毎に各種機能を実装している。

- develop: ベース機能
- feature/active_track_mission：アクティブトラックミッション
- feature/tap_fly_mission：タップフライミッション
- feature/flyzone：飛行制限空域情報の取得
- feature/waypoint：ウェイポイントミッション
- feature/status：機体ステータスの取得
